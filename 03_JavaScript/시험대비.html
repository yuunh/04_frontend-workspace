1. css 파일 => 외부스타일 연결방법 - 외부 스타일 방식 : 스타일 정보만을 따로 기술하는 .css 외부문서를 만들고 구역에 link 태그를 통해 연결시켜주는 방식 - 2. html head body - head 머리부 : 해당 문서의 각종 정보와, 제목, 설명 및 스크립트, 스타일 시트 내용 작성 - body 몸체부 : 화면상에 출력해서 보여주고자 하는 모든 정보/내용들을 작성하는 부분 3. h 태그 종류 - h1 ~ h6 : 숫자가 커질수록 크기 작아짐 4. html 기본 구조 5. 각종 태그들 - br : 줄바꿈 태그 - p : 문단 영역을 나누는 태그, 줄바꿈 하고자 한다면 별도의 태그 작성 필요, 공백은 한 개의 공백만 표시하기 때문에 별도의 기호문구 기술 - pre : 문단 영역을 나누는 태그, 시작 태그에서부터 종료 태그까지 입력된 내용을 그대로 표현함 - b : 글자를 굵게 표시하는 태그 - em : 글자를 기울여서 표시하는 태그 - mark : 형광펜 효과를 주는 태그 - u : 글자에 밑줄이 긋는 태그 - s : 글자에 취소선 넣어주는 태그 - small : 글자를 작게 표현해주는 태그 - sub : 아래첨자 / sup : 윗첨자를 나타내는 태그 - a : 하이퍼링크 걸어주는 태그 - ul : 순서 없는 목록 태그 - ol : 순서 있는 목록 태그 (start="시작값" type="숫자값, A, a, i, I") - img : src="삽입하고자 하는 이미지의 경로", alt="이미지 설명 문구" => 이미지를 보여줄 수 없을 때 해당 이미지를 대체할 텍스트 - audio : src="오디오 경로", controls(재생도구 출력 여부), autoplay(자동재생 여부 지정), loop(반복 재생 여부 지정) - video : src="비디오 경로", controls autoplay loop, poster="썸네일 이미지 경로" 6. 블럭요소 & 인라인요소 - 블럭요소 : 한 줄 단위로 영역을 차지함, 줄바꿈이 적용되어 이미 존재하는 태그의 다음줄에 영역이 잡힘 (p, pre, "div", ....) - 인라인요소 : content 영역에 해당하는 부분만 영역을 차지하는 요소, 줄바꿈이 적용되지 않아 옆으로 영역이 잡힘 (b, mark, s, img "span", ....) 7. 표 관련 태그 - table : 기본적인 표를 생성해주는 태그 - tr : 표의 한 행을 나타내는 태그 - th : 표의 제목 셀을 나타내는 태그 => 글자 굵게, 가운데 정렬 - td : 표의 일반 셀을 나타내는 태그 - colspan="" : 열 합치기 - rowspan="" : 행 합치기 8. 아이디 & 클래스 - 아이디(id) : 현재 문서 내에 특정 html 요소 하나만을 선택하고자 할 때 사용 (고유한 아이디값 부여해야 함) - 클래스(class) : 현재 문서 내에 내가 원하는 요소 여러 개를 선택하고자 할 때 사용 (중복된 값 작성 가능) 9. input type - text : 한 줄짜리 텍스트를 입력할 수 있는 텍스트 상자 (기본값) - password : 비밀번호를 입력할 수 있는 텍스트 상자 - search|url|email|phone : 검색|홈페이지|이메일|전화번호를 입력할 수 있는 텍스트 상자 - number : 숫자값만 입력 가능한 텍스트 상자 - range : 슬라이드바를 통해 숫자 지정 가능 - date : 연도, 월, 일을 입력받을 수 있는 텍스트 상자 - time : 오전/오후, 시, 분을 입력받을 수 있는 텍스트 상자 - radio : 제시한 여러 개의 값들 중에서 오로지 한 개만 선택해야 될 때 - checkbox : 제시한 여러 개의 값들 중에서 여러 개 선택 가능할 경우 - color : 색상을 선택할 수 있는 상자 - file : 첨부하고자 하는 파일을 선택할 수 있는 상자 - submit : 입력받은 데이터를 전송하는 버튼 - reset : 초기화 버튼 - button : 클릭할 수 있는 일반 버튼 - hidden : 특정값을 사용자에게 입력받지 않고, 화면에 노출시키지도 않고, 근데 넘기고자 하는 값이 있을 경우
