1. css 파일 => 외부스타일 연결방법 - 외부 스타일 방식 : 스타일 정보만을 따로 기술하는 .css 외부문서를 만들고 구역에 link 태그를 통해 연결시켜주는 방식 - 2. html head body - head 머리부 : 해당 문서의 각종 정보와, 제목, 설명 및 스크립트, 스타일 시트 내용 작성 3. h 태그 종류 - h1 ~ h6 : 숫자가 커질 수록 크기 작아짐 4. html 기본 구조 5. 각종 태그들 - br : 줄바꿈해주는 태그 - p : 문단 영역을 나누는 태그, 줄바꿈하고자 한다면 별도의 태그 작성 필요, 공백은 한 개의 공백만 표시하기 때문에 별도의 기호문구 기술 - pre : 문단 영역을 나누는 태그, 시작 태그에서부터 종료 태그까지 입력된 내용을 그대로 표현함 - b : 글자를 굵게 표시하는 태그 - em : 글자를 기울여서 표시하는 태그 - mark : 형광펜 효과를 주는 태그 - u : 글자에 밑줄이 긋는 태그 - s : 글자에 취소선 넣어주는 태그 - small : 글자를 작게 표현해주는 태그 - sub : 아래첨자 / sup : 윗첨자를 나타내는 태그 - a : 하이퍼링크 걸어주는 태그 - ul : 순서 없는 목록 태그 - ol : 순서 있는 목록 태그 (start="시작값" type="숫자값, A, a, i, I") - img : src="삽인하고자 하는 이미지의 경로", alt="이미지 설명 문구" - audio : src="오디오 경로", controls(재생도구 출력 여부), autoplay(자동재생 여부 지정), loop(반복 재생 여부 지정) - video : src="비디오 경로", controls autoplay loop, poster="썸네일 이미지 경로" 6. 블럭 속성(한줄단위) 인라인 속성(컨텐츠단위) 차이점 7. 표태그 병합 8. 아이디(고유) 클래스(여러개 가능) 차이점 9. input type = ??
